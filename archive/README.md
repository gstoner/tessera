# Tessera Archive

This directory is the single archive root for historical, superseded, and
pre-canonical material. Each subdirectory is named for the top-level area the
material came from:

- `benchmarks/` — archived benchmark concepts.
- `docs/` — pre-canonical or superseded documentation.
- `examples/` — archived examples and source drops.
- `research/` — archived research prototypes.
- `src/` — superseded production-source snapshots.
- `tests/` — historical test suites.

Archived material is retained for reference only. New active work should land
in the corresponding canonical top-level directory, not under `archive/`.
