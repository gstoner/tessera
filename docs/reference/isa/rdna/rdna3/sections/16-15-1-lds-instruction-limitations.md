# 16.15.1. LDS Instruction Limitations

> RDNA3 ISA — pages 546–546

16.15.1. LDS Instruction Limitations
Some of the DS instructions are available only to GDS, not LDS. These are:

  • DS_GWS_SEMA_RELEASE_ALL
  • DS_GWS_INIT
  • DS_GWS_SEMA_V
  • DS_GWS_SEMA_BR
  • DS_GWS_SEMA_P
  • DS_GWS_BARRIER
  • DS_ORDERED_COUNT
