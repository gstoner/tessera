15.1.2. SOPK

    Description     This is a scalar instruction with one 16-bit signed immediate (SIMM16) input and a single
                    destination. Instructions that take 2 inputs use the destination as the first input and the
                    SIMM16 as the second input.
                    E.g. "S_ADDK_I32 S0, 1" means "S0 = S0 + 1".
