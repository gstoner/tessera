# Research Archive

This folder is reserved for research prototypes that are no longer active but
may still be useful as historical context or source material.

Archived research is not part of the production build, test suite, or active
documentation surface.

Use this folder for:

- abandoned experiments
- superseded research drops
- prototypes whose ideas have been promoted elsewhere

Do not place generated files here. Delete build outputs, bytecode caches, local
metadata, and editor artifacts instead.
